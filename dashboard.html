Verification Methodology

Each API is checked hourly through direct endpoint testing. Results are logged to an immutable SQLite database with cryptographic hashing.

Chain Integrity: Each check is linked to the previous check via SHA-256 hashes, creating a blockchain-like verification chain.

Discrepancy Detection: We compare our measurements against official status pages. Variances >2% are flagged and reported.

Data Retention: All checks are stored indefinitely. The database cannot be modified retroactively.
